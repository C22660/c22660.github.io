Projet 4 : Interface utilisateur pour une application web

Posted on mar. 22 février 2022 in Projet étudiant

Pitch du projet :

Développer un site web qui permettra aux abonnés de visualiser des suggestions de films par l’intermédiaire de carrousels type « Netflix ».
Lorsque l’on clique sur l’image d’un des films une fenêtre modale s’ouvre avec des informations complémentaires.

Compétences travaillées :

  1. Interagir avec une API REST :
    Récupération des données des films depuis l’API à l’aide de requêtes ajax (réaliser des requêtes HTTP de manière asynchrone (et donc non bloquante)) et les afficher sur l’interface.
  2. Développer la partie Front-End de l’application avec HTML, CSS et JavaScript :
    Création de carrousels avec les affiches des films et de fenêtres modale comportant le détail des films.
    Aucun framework (type Bootstrap) n'est utilisé pour ce projet.

Technologies et outils :

Vanilla Java Script ; HTML ; CSS ; GitHub ; Gitbash ; PyCharm
