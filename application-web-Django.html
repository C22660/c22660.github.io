Projet 5 : Application web avec Django

Posted on dim. 27 février 2022 in Projets étudiant

Pitch du projet :

Une entreprise souhaite commercialiser un produit permettant à une communauté d'utilisateurs de consulter ou de solliciter une critique de livres à la demande.
L'objectif de ce travail est de mettre en place une application web pour leur MVP (minimum viable product, ou produit viable minimum).

Compétences travaillées :

  1. Elaborer une architecture orientée autour de trois pôles MVT (modèle, vue, template).
  2. Créer une interface utilisateur correspondant à celle conçue dans les wireframes, s'affichant côté serveur à l'aide du langage de template Django.
  3. Tenir compte des exigences énoncées dans le cahier des charges, et présenter une structure de base de données équivalente à celle du schéma de la base de données.
  4. Rédiger, via le README un ensemble d'instructions détaillant la configuration de l'environnement et le fonctionnement de l'application.

Technologies et outils :

Python 3.9 ; Django ; Pillow ; Sqlite3 ; GitHub ; Gitbash ; PyCharm
