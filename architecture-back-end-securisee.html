Projet 8 : Architecture back-end sécurisée

Posted on dim. 27 février 2022 in Projet étudiant

Pitch du projet :

Une société d’événementiel souhaite de doter d’un système CRM (Customer Relationship Management ou gestion de la relation client) sécurisé interne à l'entreprise.

Compétences travaillées :

  1. Rédiger le code d'une application présentant des points de terminaison d'API sécurisés utilisant le framework Django REST pour permettre les opérations CRUD.
  2. Mettre en œuvre une base de données sécurisée avec Django ORM et PostgreSQL :
    Elaboration de l'architecture d'une base de données relationnelle par un diagramme entité relation Configuration dans le fichier « settings.py » d’une base de données PostgreSQL
  3. Effectuer le logging et la surveillance
  4. Personnaliser le site d'administration Django en tant qu'interface front-end

Technologies et outils :

Python 3.9 ; Django ORM ; PostgreSQL ; GitHub ; Gitbash ; PyCharm ; Draw.io
